# previsao
Aplicação web desenvolvida com HTML, CSS e JavaScript que consome a API do OpenWeatherMap. O projeto realiza buscas automáticas de dados meteorológicos, apresentando temperatura, umidade, vento e um relógio dinâmico sincronizado com o fuso horário da região. Possui uma interface inteligente que adapta seu esquema de cores com base no clima e na hora do dia.
